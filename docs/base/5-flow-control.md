# Flow Control

Controlling code execution paths

---

## Conditional Blocks

Follow the form:
```
if expr {
	
} else if expr {
	
} else {
	
}
```

---

## Switch Blocks

Follow the form:
```
switch expr {
case tag1:
    // handle expr == tag1
case tag2, tag3:
    // handle expr == tag2 || expr == tag3
else:
    // handle all other cases
}
```

---

## Loops

Follow the form:
```
loop [assign;] expr[; expr] {
	
}
```
